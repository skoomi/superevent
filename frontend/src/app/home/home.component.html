Aplikacja do zarządania eventami

Aplikacja dla niezalogowanego użytkownika pozwala przeglądać organizowane eventy wraz z ich szczegółami.
Po zalogowaniu użytkownik moze zapisać się do wybranego eventu, przeglądać wszystkie do których już jest zapisany oraz złożyć podanie o wypisanie z wybranego.
Pracownik po zalogowaniu może tworzyć i kasować eventy, wypisywać i przenosić uczestników. Może również blokować możliwość zapisów.
Administrator może dodatkowo tworzyć nowe konta z uprawnieniami procownika.

Wykorzystane technologie:
  • Java
  • Angular
  • Spring Boot
  • Spring Security
  • MySQL
  • JPA
  • HTML
  • Bootstrap
